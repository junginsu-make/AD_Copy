## 🆘 긴급 문제 해결

### 앱이 실행되지 않아요
```bash
# 1. PM2 로그 확인
pm2 logs ad-copy-generator

# 2. 환경 변수 확인
cat .env.local

# 3. 빌드 재실행
npm run build
pm2 restart ad-copy-generator
```

### 웹에서 접속이 안 돼요
```bash
# 1. Nginx 상태 확인
sudo systemctl status nginx

# 2. Nginx 로그 확인
sudo tail -f /var/log/nginx/error.log

# 3. 방화벽 확인 (AWS Security Group)
# EC2 대시보드에서 포트 80, 443 확인
```

### API가 작동하지 않아요
```bash
# 1. 환경 변수 확인
nano .env.local

# 2. API 키 테스트
# Supabase, OpenAI, Gemini 등 키가 유효한지 확인

# 3. 로그 확인
pm2 logs ad-copy-generator --lines 50
```
